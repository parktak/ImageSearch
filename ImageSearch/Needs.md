
1. 과제 내용
있는 앱을 구성합니다.
: 단일 화면에서 검색어 입력 필드와 API 검색 결과 이미지 목록 + 북마크 이미지 목록을 볼 수

[제약조건]
• Language: Swift 6.0 이상
• Deployment Target: iOS 17.0 이상
• Framework: SwiftUI
• Zero External Dependency:
- Network: URLSession 을 직접 래핑한 Generic 네트워크 레이어 구현합니다.
- Image Loading: 자체 Image Downloader를 구현합니다.
- DI: 의존성 주입을 위한 자체 구조 설계합니다.
• 형상관리 (Git): 작업 과정이 드러나는 커밋 히스토리를 포함합니다.
• Kakao 이미지검색 API를 사용하며, 검색 결과는 JSON 방식으로 받습니다.
- API Key는 개발자 본인이 직접 발급받아 사용합니다.


2. 구성요소 설명
1) 메인 화면 상단
• 검색어 필드
- 검색 버튼은 별도로 존재하지 않습니다.
- 1초 이상 검색어 입력이 없을 경우 검색을 수행합니다.
- 검색 결과가 없거나 오류가 발생한 경우 이를 사용자에게 알려주어야 합니다.

2) 메인화면 하단 탭
• 탭1: API 검색 결과 이미지 목록
- 검색 결과로 표시되는 이미지의 가로 크기는 화면 폭과 동일하게 표시되며, 세로 크기는 이미지 비율에 맞게 조정되어야 합니다.
- 검색결과엔 북마크가 가능하도록 버튼 UI를 추가합니다.
- 북마크한 경우 별도로 체크표기를 노출하고, 선택 or 해제할 수 있어야 합니다.

• 탭2: 북마크 이미지 목록
- 사용자가 북마크한 이미지 목록을 표기합니다.
- 북마크된 데이터의 표기조건은 검색결과를 표기할 때와 같아야 합니다.
